# PIC_board_starter_pcb
this is a tutorial from Waterloo Rocketry to demonstrate the PIC board and the KiCad workflow

## Process include
1. create custom symbol for PIC12(L)F1501
2. create Schematic
3. create PCB layout  
4. Generate Gerber and Drill File
